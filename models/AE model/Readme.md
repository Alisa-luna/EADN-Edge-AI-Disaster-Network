# 自编码器（AE）异常检测模块

## 概述

自编码器（Autoencoder, AE）是节点端地震预警的第一步。它不直接识别地震波形，而是学习**本台站当前环境的正常振动模式**。当环境发生显著改变（如地震波到达）时，AE 的重建误差、潜在向量距离等指标会超出基线范围，从而触发后续的 AI 确认与参数估算流程。

这种“检测偏离正常”的方式，使得系统能够适应不同台站的噪声特性（安静郊区、繁华街道、工厂振动等），不需要预先采集大量地震样本。

## 模型结构

AE 采用轻量级卷积自编码器，适合在 ESP32‑S3 上实时运行：

- **输入**：6×200 的二维窗口（6 通道 × 200 个采样点，100 Hz 下 2 秒）
  - 通道 0‑2：去均值后的 Z、N、E 三轴加速度
  - 通道 3‑5：Z、N、E 的滑动 RMS 包络（简化希尔伯特包络）
- **编码器**：三层 Conv1d（6→16→32→64，stride=2）+ 全局平均池化 → 16 维潜在向量
- **解码器**：全连接层 + 三层转置卷积（64→32→16→6），输出重建的 6×200 窗口
- **输出**：重建窗口、潜在向量，以及三个异常指标：
  - **高频重建误差**：输入与输出相邻采样点差分的 MSE（对规律性信号更敏感）
  - **潜在向量距离**：潜在向量的 L2 范数
  - **频谱幅值误差**：各通道平均绝对幅值之差

三个指标加权融合为一个**综合异常分数**，用于与基线比较。

## 异常检测流程

1. **环境学习**  
   每到一个新小时，节点静默收集 **30 个连续窗口**（约 30 秒）的综合异常分数。对这 30 个分数排序，取中位数和最小/最大值，并按照：
   - 下界 = 中位数 − (中位数 − 最小值) × 系数
   - 上界 = 中位数 + (最大值 − 中位数) × 系数  
   建立该小时的聚类基线。默认系数为 1.0，可根据灵敏度调整。

2. **实时监测**  
   基线建立后，每个新窗口的综合分数若 **低于下界** 或 **高于上界**，即判定为异常。  
   实际应用中，地震波到达时高频误差通常会骤降（信号变得规律），同时潜在向量距离和频谱误差骤升，综合分数可能跌至正常范围外。

3. **触发后续链**  
   AE 异常仅表示“环境改变”，不直接认定地震。它会立即调用 **Edge Impulse CNN 模型** 对历史 Z 轴数据进行分类，若分类为地震（置信度 > 0.5），则进一步通过 **双编码器神经网络** 估算 τc、S‑P 走时、峰值加速度及 P 波方位角，并将结果打包成 37 字节帧通过 LoRa 发送。

## 权重训练

AE 的权重在部署前通过 Python 离线训练获得，不进行在线更新。

- **训练数据**：节点端通过串口导出 6 通道归一化窗口，在 PC 上累积数小时正常环境数据。
- **模型定义**：见 `models/train_ae.py`，结构与 ESP32 推理代码一致。
- **损失函数**：仅 MSE 重建损失（无需频域约束），batch size = 64，Adam 优化器，100~200 epoch。
- **导出**：训练好的权重通过 `export_weights.py` 转换为 C 头文件 `ae_weights.h`，编译进节点固件。

## 基线管理

- 每个小时独立维护一条基线（`AEHourBaseline`），包含中位数、下界、上界。
- 基线仅在**当前小时的前 30 个窗口**内建立，之后冻结，不再更新，避免学习到异常信号。
- 断电后基线通过 Preferences 存储，重启后若发现已冻结则直接使用，否则重新学习。
- 若基线值全为 0（旧版本残留），会被自动清除并重新采集。

## 主要函数

| 函数 | 说明 |
|------|------|
| `ae_predict()` | 前向推理，返回重建输出、潜在向量及各项误差 |
| `ae_update_baseline(hour, score)` | 收集综合分数，达到 30 个后建立基线 |
| `ae_get_threshold(hour)` | 返回当前小时的异常判定范围（下界/上界） |
| `ae_save_baseline()` / `ae_load_baseline()` | 基线的持久化读写 |

## 参数调优

| 参数 | 位置 | 说明 | 默认值 |
|------|------|------|--------|
| `BASELINE_SAMPLES` | `ae_model.cpp` | 建立基线需要的窗口数 | 30 |
| `BASELINE_RANGE_K` | `ae_model.cpp` | 范围扩大系数（越小越敏感） | 1.0 |


## 注意事项

- 节点安装后需静置至少 30 秒，让 AE 学完当前小时的基线。
- 若环境发生永久性改变（如搬动节点），需重启节点或等待下一小时重新学习。
- AE 仅做异常检测，不输出地震参数，需配合 CNN 和双编码器完成完整预警。
