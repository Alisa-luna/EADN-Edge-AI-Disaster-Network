**过去的模型**

这个文件夹包含原先尝试过的TCN/CNN时序预测模型。

*但在目前的实验条件下，可在单片机上运行且具有实用性时序预测模型均会受到当地台站噪声的影响，信号重建corr<<噪声corr*

项目也曾尝试过用多种更先进的神经网络架构（包括但不限于Transformer/CNN/LSTM/FCN/VAE/U-Net/多种架构混合模型，参数量100k~500k不等），

用了多种训练方法(包括但不限于lr学习率调度,分级训练，Optuna网格扫描)

（注：共训练模型125+3=128个模型，总耗时近3个月）


但最终结果显示，模型学到的更多是噪声而非信号。


噪声 vs 信号 最终对比

信号段: 103个, Corr均值=0.343, 中位=0.251

噪声段: 23个, Corr均值=0.594, 中位=0.716

最佳验证Corr中位: 0.416

模型未能区分信号和噪声


因此可认为，

*在数据量不够的情况下（指总时长小于10h的训练数据情况下），输入<1s的情况下，轻量级（参数<500k）的深度学习模型几乎不可能完成对未来3s以上、具备实用价值的地震波形预测任务*

（即使是进行严格的输入输出特征检查，也很难达到实用标准，可区分度不高）

因此，对于IOT系统，项目最终选择的是放弃对波形的预测，转而对几个关键参数进行预测（即Dual_PhyNet混合模型）


**注意：这些模型已经不再也不建议使用，仅保留作为项目进展的证明**
